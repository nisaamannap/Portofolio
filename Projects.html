Product Data Migration System: E-Katalog to INAPROC Automation

RPA Implementation & Imgur Image Conversion for PT. SAE

Developed an automated workflow to streamline the migration of product data from E-Katalog to INAPROC systems. This system features automated Imgur image conversion to ensure data compatibility and efficiency.

Tech Stack: Ui.Vision RPA, HTML/JavaScript, Imgur API Integration
Core Result: Reduced manual data entry time by automating repetitive migration tasks and image handling.
